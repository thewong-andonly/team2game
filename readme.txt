This is a text based adventure game created by team two during our three week Code Nation course. Please enjoy the game.

How to play:
1.Install requirments by running > pip install -r requirements.txt
2.If 1 is too hard for you just run > pip install playsound
3.run zerona.py in terminal!
4.Have fun!
